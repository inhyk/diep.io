# diep.io 프로토타입

`diep.io` 스타일 아레나 슈터를 브라우저만으로 구현한 무의존성 프로토타입입니다.

## 실행

브라우저에서 [index.html](./index.html)을 바로 열면 됩니다.

## 조작

- `WASD` 또는 방향키: 이동
- 마우스: 조준
- 마우스 왼쪽 버튼 유지 또는 `Space`: 발사
- `1-6`: 업그레이드 포인트 사용
- `7-9`: 가능한 전차 분기 선택
- `R`: 사망 후 재시작

## 현재 MVP

- 넓게 스크롤되는 아레나에서 진행되는 탑다운 탱크 전투
- 체력, 충돌 피해, 경험치 보상을 가진 중립 도형
- 일반 네모, 찌그러진 네모, 삼각형, 오각형, 육각형, 알파 오각형 다수
- 매치 진행에 따라 강해지고 시간이 지나며 레벨업하는 로그 적 탱크
- 시작 오버레이의 닉네임 입력과 실시간 리더보드
- 시작 화면에서 `자유전`, `땅따먹기`, `팀전`, `축구` 모드와 다수의 맵 프리셋 선택 가능
- 진행도 테스트와 소환 확인용 개발자 기능이 있는 비밀번호 잠금 제작자 모드
- 플레이어 메시지에 적 탱크가 반응해 답장하는 로컬 인게임 채팅
- 직접 가진 오디오 파일을 불러와 재생할 수 있는 로컬 배경음악 업로드
- 경험치, 레벨업, 확장된 스탯 상한, 5레벨마다 보너스 포인트, 전차 분기, 미니맵, 컴팩트 HUD
- 모바일 플레이용 터치 조이스틱

## 제작자 모드

- 비밀번호: `hihi`

## 게임 모드

- `자유전`: 기존 방식의 개인전 모드
- `땅따먹기`: 돌아다니며 지나간 타일을 팀 색으로 칠해 땅을 넓히는 모드
- `팀전`: 블루 팀과 레드 팀이 처치 점수로 싸우는 모드
- `축구`: 공을 밀어 상대 골문에 넣는 모드

## 맵 프리셋

- 현재 `클래식`, `미러`, `이클립스`, `포지`, `오아시스`, `프로스트라인`, `리프트`, `헤일로`, `시타델`, `델타`, `볼트`, `크로스윈드`, `듄스`, `펄스`, `리액터`, `아틀라스`, `바솔트`, `스톰`까지 18개 맵을 선택할 수 있습니다.
- 맵마다 배경 색감, 격자 밀도, 팀 스폰 위치, 땅따먹기 시작 패턴, 팀전 전선 모양, 축구장 방향과 골문 크기, 도형/봇 밀도가 달라집니다.
